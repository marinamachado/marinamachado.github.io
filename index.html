Our Project

While going through the available data, we noticed something: we couldn't easily find what we were looking. And that was it, and we had an idea! What about organizing the information and make it available to be accessed in a simpler way? So we developed an open-source API to gather NASA, JAXA and ESA datasets and used it to build this website.

API

Our API has a series of databases offered by space agencies, specifically ESA, JAXA, and NASA, facilitating access and manipulation of this data by interested data scientists, programmers, and laypeople. At the highest level, the covidata has three subsections, one for each of the agencies, and from these subsections, it is possible to access the databases,
e.g.:
from covidata import nasa
dataset = nasa.Landslides ( )

Each dataset is downloaded from the internet unless it is in the cache and then loaded into a panda Dataframe, one of the main python data analysis libraries. Finally, they also have a summarize method, which can be used to view summary statistics from our dataset.
